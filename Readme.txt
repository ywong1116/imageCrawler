This image crawler is written by Tsz Yang Wong.

This application uses the PHP Simple HTML DOM Parser from sourceforge, Google Drive API
and composer.

## Key features
Extracts product list from json data
Uploads extracted products onto Google drive parent folder

## Usage
Run application on command prompt with PHP by running the file "ImageCrawler.php" at
it's directory.

To retrieve the images from another website, modify lines 9, 12 and 39 to properly
point to the product locations.
